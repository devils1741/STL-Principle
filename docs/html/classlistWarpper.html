Detailed Description

template<typename T>
class listWarpper< T >

一个自定义的类，封装了stl中的list的部分功能
list是一个双向链表
节点结构有_List_node、_List_node_header以及_List_iterator和_List_const_iterator
struct _List_node_base { _List_node_base* _M_next; _List_node_base* _M_prev;}
_List_node_header继承自_List_node_base， 新增了_M_size用于表示链表的大小
